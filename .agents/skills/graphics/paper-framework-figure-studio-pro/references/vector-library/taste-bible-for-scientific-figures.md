# Taste Bible for Scientific Framework Figures

Good paper figures are clear, restrained, and structured. They are not posters.

## Positive taste rules

- One figure, one main sentence.
- Strong visual hierarchy before decorative style.
- Stage outputs are more useful than dense formulas.
- White space is a feature.
- Arrows carry typed meaning.
- Icons are landmarks, not decoration.
- Caption complements the figure; it does not rescue a confusing figure.

## Anti-patterns

- Too many formulas.
- Too many tiny icons.
- All modules equal weight.
- Color without semantic meaning.
- Soft-3D that obscures topology.
- Dense poster boards.
- Raw screenshots pasted into vector figures.
- Only-readers-of-the-paper symbols.
