# Evidence Map Index

This skill uses the existing project-local diagram evidence corpus. The full local corpus is not bundled; this package keeps a summary of the evidence artifacts and derived taxonomy.

Primary evidence sources summarized during construction:

- paper cards;
- figure inventory;
- caption inventory;
- label summary;
- panel-structure notes;
- visual-pattern observations;
- extraction report and summary;
- representative rendered pages and contact sheets;
- retrieval manifest;
- acquisition report.

Use the embedded taxonomy and label summaries before routing a new target. The labels are multi-label, so a single paper or diagram may count under multiple classes.

Representative rendered pages and contact sheets are audit aids only. Use the evidence-lineage summary, source-corpus notes, taxonomy references, and review rubric as the runtime evidence base.
